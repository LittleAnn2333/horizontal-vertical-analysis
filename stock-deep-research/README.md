# Stock Deep Research Skill (v3)

一个用于上市公司深度研究的 Claude skill,产出结构化 Word 研究报告。

## 简介

融合两套方法论:
- **横纵分析框架**:Saussure 语言学的"历时-共时"思想 + 商学院案例研究法。强项是"为什么这家公司长成今天的样子"的因果解释力。
- **金融视角**:卖方研究的财务/产业链分析模块。强项是"账本上到底多少"的数据严谨度。

最终产出兼有叙事力与数据密度的 Word 研究报告(18-25 页)。

## v3 升级要点(对标专业卖方研究的 6 项改造)

1. **5 个关键决策时刻清单化**:纵向分析末尾用 H3 独立小节列出"决策一/二/三/四/五",每个解释"为什么这一步是不可逆的锁定"
2. **产业链位置独立小节**:横向分析加上"上游/下游/位置价值"三要素分析,而不只是讲对手
3. **核心竞争力与护城河独立成章**:每个护城河按"类型/性质判定(结构性 vs 阶段性)/盈利变现路径/持续性展望/可验证证据"五要素展开
4. **三剧本必须给估值锚算式**:每个剧本必须给"PE × EPS"或"PB × 净资产"的具体算式,让读者可调试
5. **短期 vs 长期逻辑分段**:横纵交汇里加上"短期 1-3 个月催化"和"长期 1-3 年趋势"两段,给不同投资风格的读者
6. **数据缺口显性标注**:正文里直接说"现有资料无法精确推算",不遮掩、不编造

## v3 之前已有的特性

- 素材体检分级(Tier 1/2/3 严格度)
- 分析骨架预览(在写完整报告前给用户确认)
- 信源打折清单 5 列(含 🔴 支柱 / 🟡 辅助 / 🟢 背景 重要性标签)

## 报告结构(9 章)

1. 一句话定义
2. 纵向分析(阶段叙述 + 5 个关键决策时刻)
3. 横向分析(产业链位置 + 三层对手 + 估值对照)
4. 业务与财务底盘
5. 核心竞争力与护城河(每条按五要素展开)
6. 横纵交汇(决策路径 + 短期/长期逻辑 + 三剧本+估值算式)
7. 调研提纲与跟踪指标
8. 风险提示
9. 信源与打折清单

## 工作流程

```
Step 1: 素材体检(Tier 分级) → Step 2: 公开信息补全 → 
Step 3: 内部结构化分析 → Step 3.5: 分析骨架预览(给用户确认) →
Step 4: 生成 Word 报告
```

## 安装

### Claude.ai 网页端
1. 把整个 `stock-deep-research/` 文件夹打包成 .skill 文件
2. 在 Customize → Skills 上传

### Claude Code
```bash
cp -r stock-deep-research ~/.claude/skills/
```

## 依赖

- Node.js 18+
- npm 全局安装 docx 库: `npm install -g docx`

## 设计原则

- **不出具体目标价**——给三剧本概率分布,但 v3 起每个剧本必须配估值算式
- **不假装客观**——打折清单 + 正文显性标注双重透明
- **保留叙事感 + 数据密度**——避免变成纯数据堆砌或空泛软文
- **让用户参与推理**——骨架预览让用户在写完整报告前介入

## 兼容性

v3 所有新字段都是**可选**的(老数据无修改可继续跑),但新模式 A 报告**应该**全部填上,这是 v3 价值的核心体现。
